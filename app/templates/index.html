Clasificación automática de productos

Sube una imagen de ropa, calzado o accesorios. El sistema la analizará con Amazon Rekognition, asignará una categoría automática y guardará el resultado en la base de datos.

Subir imagen de producto

1. Carga

El usuario sube una imagen desde la interfaz web.

2. Almacenamiento

La imagen se guarda en un bucket privado de Amazon S3.

3. IA

Amazon Rekognition detecta etiquetas relevantes.

4. Resultado

El sistema asigna una categoría y guarda el resultado.
